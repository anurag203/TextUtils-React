# TextUtils - Edit text online.
## **Tech Stack** : 
* HTML
* CSS
* JavaScript
* React Js
* Bootstrap
* Git / GitHub
___
## **Description** : 
* Easy to Use
* Offers various features
* Fast and Error free editing
* Preview Text file
* Shows charcter and Word count
* Dark mode available
* Completely free
